# Abrir Ubuntu desde VSCode local

Este proyecto vive realmente en Ubuntu, y esta copia local sirve como respaldo y como punto de entrada desde VSCode.

## Host SSH listo
- Alias: `plataformacorporativa-ubuntu`
- Host: `192.168.1.40`
- Usuario: `joseph`

## Pasos
1. Abre VSCode local.
2. Instala la extensión `Remote - SSH`.
3. Ejecuta `Remote-SSH: Connect to Host...`.
4. Elige `plataformacorporativa-ubuntu`.
5. Abre la carpeta remota `/home/plataformacorporativa`.

## Qué vas a ver
- La carpeta real del proyecto en Ubuntu.
- Los mismos archivos de trabajo.
- El terminal del servidor para probar y ejecutar cambios.

## Regla
- Edita en local.
- Ejecuta y valida en Ubuntu.
- Mantén esta carpeta local como respaldo.
