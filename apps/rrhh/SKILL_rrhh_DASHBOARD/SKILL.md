# SKILL_rrhh_DASHBOARD

## Proposito
Define el dashboard de RRHH con headcount, ausentismo, rotacion, horas, cumplimiento y alertas.

## Debe contener
- KPI de headcount
- KPI de ausentismo
- KPI de rotacion
- KPI de horas trabajadas
- KPI de cumplimiento documental
- origen de cada indicador
- reglas de refresco

## Contexto obligatorio
- recibe `usuario`
- debe mostrar solo datos autorizados
- debe indicar fuentes de cada indicador

## Salida esperada
- cards de talento
- graficas de asistencia
- tablas de alertas
- consultas o vistas de soporte
