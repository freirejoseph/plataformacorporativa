# SKILL_produccion_DASHBOARD

## Proposito
Define el dashboard de produccion con OEE, cumplimiento, productividad, retrasos y alertas de planta.

## Debe contener
- KPI de OEE
- KPI de cumplimiento
- KPI de productividad
- KPI de scrap
- KPI de paradas
- origen de cada indicador
- reglas de refresco

## Contexto obligatorio
- recibe `usuario`
- debe mostrar solo datos autorizados
- debe indicar fuentes de cada indicador

## Salida esperada
- cards de planta
- graficas operativas
- tablas de excepciones
- consultas o vistas de soporte
