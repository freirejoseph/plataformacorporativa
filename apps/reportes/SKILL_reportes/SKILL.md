# SKILL_reportes

## Proposito
Define el catalogo de reportes corporativos, sus fuentes, permisos, refresco y versionado.

## Alcance
- Administrar catalogo de reportes.
- Definir permisos de acceso por rol y usuario.
- Registrar fuentes de datos y frecuencia de actualizacion.
- Versionar cambios y exportaciones.

## Debe contener
- catalogo de reportes y carpetas
- fuentes de datos y dependencias
- reglas de permiso y acceso
- validaciones de publicacion
- integracion con exportacion y auditoria

## Contexto obligatorio
- recibe `usuario`
- debe respetar permisos del portal
- no debe exponer datos fuera del alcance del usuario

## Salida esperada
- estructura de datos
- endpoints o consultas base
- reglas de integracion con el portal principal
