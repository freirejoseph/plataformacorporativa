# SKILL_reportes_DASHBOARD

## Proposito
Define el dashboard del catalogo de reportes con uso, favoritos, errores, tiempo de respuesta y refresco.

## Debe contener
- KPI de reportes usados
- KPI de favoritos
- KPI de errores
- KPI de tiempo de respuesta
- KPI de reportes vencidos por refresco
- origen de cada indicador
- reglas de refresco

## Contexto obligatorio
- recibe `usuario`
- debe mostrar solo datos autorizados
- debe indicar fuentes de cada indicador

## Salida esperada
- cards de uso
- graficas de frecuencia
- tablas de soporte
- consultas o vistas de analitica
