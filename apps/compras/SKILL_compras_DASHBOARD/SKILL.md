# SKILL_compras_DASHBOARD

## Proposito
Define el dashboard de compras con indicadores de abastecimiento, aprobaciones, proveedores y costo.

## Debe contener
- KPI de ordenes abiertas
- KPI de recepciones pendientes
- KPI de proveedores activos
- KPI de ahorro vs presupuesto
- KPI de cumplimiento OTIF
- origen de cada indicador
- reglas de refresco

## Contexto obligatorio
- recibe `usuario`
- debe mostrar solo datos autorizados
- debe indicar fuentes de cada indicador

## Salida esperada
- cards de control
- graficas de seguimiento
- tablas de excepciones
- consultas o vistas de soporte
