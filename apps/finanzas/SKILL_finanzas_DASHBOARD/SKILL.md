# SKILL_finanzas_DASHBOARD

## Proposito
Define el dashboard financiero con ventas, gastos, margen, liquidez y variaciones por periodo.

## Debe contener
- KPI de ventas
- KPI de gastos
- KPI de margen
- KPI de liquidez
- KPI de variacion presupuestaria
- origen de cada indicador
- reglas de refresco

## Contexto obligatorio
- recibe `usuario`
- debe mostrar solo datos autorizados
- debe indicar fuentes de cada indicador

## Salida esperada
- cards financieras
- graficas de tendencia
- tablas de variacion
- consultas o vistas de soporte
