# SKILL_administracion-general_DASHBOARD

## Proposito
Define el dashboard de gobierno corporativo con estado de parametros, licencias, politicas y tareas transversales.

## Debe contener
- KPI de parametros configurados
- KPI de licencias vigentes
- KPI de politicas activas
- KPI de tareas pendientes
- origen de cada card o indicador
- filtros por usuario o rol
- reglas de refresco

## Contexto obligatorio
- recibe `usuario`
- debe mostrar solo datos autorizados
- debe indicar la fuente de cada indicador

## Salida esperada
- cards de resumen
- tablas de control
- graficas simples
- consultas o vistas de soporte
